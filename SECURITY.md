# 安全政策 (Security Policy)

## 受支持的版本

我们目前仅直接支持以下版本的安全更新：

| 版本 | 是否支持 |
| :--- | :--- |
| v34.5 (Flagship) | ✅ 是 |
| v16.x | 🛑 否 |

## 报告漏洞

我们非常重视系统的安全性。如果您发现了任何安全漏洞（尤其是涉及 API Key 泄露或文件系统越权访问风险），请按照以下步骤报告：

1.  **请勿** 在 GitHub Issue 中公开披露漏洞。
2.  请将详细的漏洞描述、复现步骤及受影响的组件发送邮件至作者邮箱（见配置文件）。
3.  我们将在 48 小时内确认并评估风险。

感谢您为保护本项目安全性所做的贡献。
