## ⚡️ 极速初始化引擎
遵循以下工业标准流，在 3 分钟内唤醒你的本地创作中枢。
<Steps>
1.  **唤醒底层看门狗**
    在你的项目根目录执行以下命令，启动本地开发服务器与 AI 嗅探引擎：
    ```bash
    npm run dev
    ```
2.  **创建你的第一篇双链笔记**
    在 `src/content/docs/` 中建立任意 `.md` 文件，尽情使用你的编辑器特性。
3.  **构建生产级静态资产**
    当创作就绪，执行物理歼灭与全量静态构建：
    ```bash
    npm run build
    ```
</Steps>
