# 📅 Illacme Plenipes 迭代里程碑计划 (V35.0 - V40.0)

## ⛳ 阶段一：奠基 - 主权中枢与准入卫士 (Foundry Foundation)
*   **核心目标**：实现出版社的物理隔离与授权红线。
*   **关键任务**：
    1.  重构 `WorkspaceManager`：支持目录化镜像与生命周期管理。
    2.  强化 `LicenseGuard`：实现基于 SIG 证书的权益位校验。
    3.  升级 `ConfigManager`：支持多出版社配置的热加载。

## ⛳ 阶段二：启航 - 自动化向导 (The Foundry Voyage)
*   **核心目标**：大幅降低准入门槛，实现“分钟级”开店。
*   **关键任务**：
    1.  开发 `WebWizard` 容器：基于本地临时 Web 端口的引导界面。
    2.  算力自动探测：智能识别 Ollama/LM Studio 节点并测试连通性。
    3.  自动拓扑生成：根据 SSG 主题自动配置 i18n 物理路径。

## ⛳ 阶段三：精研 - 编辑方言与精准收稿 (Editorial Refinement)
*   **核心目标**：提升出版的专业深度与灵活性。
*   **关键任务**：
    1.  实装编辑方言模板系统：支持风格文件的镜像与自定义。
    2.  开发“发布路由矩阵”：实现 Markdown 子目录到 SSG 路径的精准映射。
    3.  升级影子缓存库：实现基于 Hash 的纯净资产管理。

## ⛳ 阶段四：洞察 - 总编室仪表盘 (The Newsroom Dashboard)
*   **核心目标**：实现出版全流程的可视化与实时干预。
*   **关键任务**：
    1.  开发 Dashboard 后端建立全链路事件流（Event Stream）。
    2.  可视化视图开发：进度流转图、算力消耗分布、多语种关系图谱。
    3.  翻译管理：实现在界面端开启、暂停与调整翻译风格。

## ⛳ 阶段五：连接 - 全球发售与联合供稿 (Global Syndication)
*   **核心目标**：打破数据孤岛，实现内容的全网覆盖。
*   **关键任务**：
    1.  开发分发渠道适配器：Webhook, S3, 以及微信、掘金等社交插件。
    2.  健康度自愈审计：全面覆盖本机、算力、发布、分发等环节的运行监控。
    3.  商业化闭环：完善授权激活流与权益提示。
