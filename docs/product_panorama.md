# 🏛️ Illacme Plenipes 产品全景设计大纲

## 1. 核心品牌主权 (Branding)
*   **定位**：您的全球私人出版社 (Your Global Private Press)。
*   **理念**：数据主权、算力民主、物理隔离、出版尊严。

## 2. 五层分层架构 (Layered Architecture)
### 第 I 层：主权空间 (Sovereign Workspace)
*   **租户隔离**：每个 Workspace 是一个独立的文件夹，包含自己完整的“生命特征”（配置、账本、装帧、日志）。
*   **物理版图**：`workspace/[press_name]/` 包含该社所有资产。

### 第 II 层：原稿仓库 (Manuscript Warehouse)
*   **多源接入**：支持本地磁盘、远程 Git、云端 API 等多种“收稿渠道”。
*   **发布切片**：支持从根目录发布（免费版）或指定子目录映射（授权版）。

### 第 III 层：编辑方言 (Editorial Dialects)
*   **风格模板**：基于提示词工程的“翻译风格”文件，支持从核心库镜像到空间进行微调。
*   **颗粒度控制**：支持按目录分配不同的编辑风格。

### 第 IV 层：影子缓存 (Universal Shadows)
*   **算力保存**：存储不带标签的“纯净译文”，按 `Style_Hash` 隔离，实现跨主题的算力复用。
*   **内容完整性**：仅负责正文与 Frontmatter 的多语种转化。

### 第 V 层：装帧工场 (The Bindery)
*   **主题热切换**：主题从主程序镜像到空间，支持 Starlight (自编译型) 与 Default (内置型)。
*   **自动拓扑**：启用主题时自动构建 i18n 目录结构与资源搬运。

## 3. 商业权益矩阵 (Commercial Tiers)
*   **社区版 (Free)**：
    - 仅限 1 个出版社实例。
    - 仅限原稿库根目录发布。
    - 仅限 1 种目标语。
    - 全局仅限 1 种翻译风格。
*   **主权版 (Professional)**：
    - 无限出版社实例。
    - 支持子目录精准筛选与 SSG 映射。
    - 支持全语种矩阵。
    - 支持按目录定制编辑风格。
