# 🌟 Illacme-plenipes 核心特性宪章 (Core Features Charter)

本文件定义了 Illacme-plenipes 引擎的核心功能目标与设计哲学，是所有架构演进与功能迭代的最高准则。

---

## 🚀 十大核心能力

### 1. 本土创作，全球传播
用户只需使用最熟悉的本土语言进行 Markdown 创作，引擎全自动为用户实现全球化分发。

### 2. 极致的 Markdown 兼容性
不限制用户的 Markdown 文档库方言，支持标准格式及常见扩展格式（如 Obsidian/Logseq 语法），实现任意格式文档的兼容解析。

### 3. 全链路 AI 算力适配
支持各类云端大模型 API（OpenAI, Gemini, DeepSeek 等）以及本地大模型 API（Ollama, LM Studio）调用，支持差异化参数配置与推理开关控制。

### 4. 语义化自动翻译
通过大模型全自动将 Markdown 文件翻译为多国语言版本，支持基于语义的分片（Slicing）及分段（Chunking）翻译，确保长文不丢失上下文。

### 5. 自动化 SSG 适配
支持任意 SSG 框架（Docusaurus, Starlight, Hugo 等）作为项目主题。自动适配 SSG 默认文档目录与静态资源保存路径，实现“一套源码，多端发布”。

### 6. 全面 SEO 优化治理
- **Slug 转换**：根据文件名自动处理为标准英文版 URL-Safe Slug。
- **元数据提炼**：自动分析提炼关键词（Keywords）、描述（Description）、标签（Tags）与分类（Categories），并按 SSG 规范补全 Frontmatter。

### 7. 智能资产流水线
全自动提取文档中引用的附件、图片等资源，保存到 SSG 静态目录，并自动执行 Slug 化命名替换，确保路径的相对稳定性。

### 8. 自动化关系图谱 (Knowledge Graph)
全自动生成文档库的关联图谱，支持在所有 SSG 页面中无缝插入，并提供全屏显示支持。

### 9. 自动化门户生成
为每种 SSG 主题生成功能介绍首页、操作指南与导航入口。根据主题特点自动生成顶部菜单、分类列表及示例页面。

### 10. 工业级架构设计
- **分层解耦**：插件化热拔插设计。
- **高并发**：支持多模型并行调度与主备切换模式。
- **全方位监控**：内置提示词自定义、看门狗监控、同步/翻译进度追踪、Token 消耗统计及多级日志控制。

---

> [!IMPORTANT]
> **设计哲学：让创作者回归创作本身。**
> 任何功能迭代如果增加了创作者的操作负担，或破坏了 Markdown 的开放性，均应被视为违背本宪章。
